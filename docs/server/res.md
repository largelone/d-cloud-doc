# 领地保护系统

我们使用领地插件来保护您的财产和建筑。

## 创建领地教程

现在开始教学如何创建一个领地。

### 步骤说明

1. **选择初始点**：首先，这是我们的示例区域
   
   ![选择初始点](/guide/res/create/1.png)

2. **设置起始位置**：使用鼠标左键选择一个方块作为初始点
   
   ![设置起始位置](/guide/res/create/2.png)

3. **设置结束位置**：使用鼠标右键选择终止点
   
   ![设置结束位置](/guide/res/create/3.png)

4. **形成区域**：两点之间立体自然形成方形区域

5. **完成创建**：输入 `/res create <名字>` 完成创建
   
   ![完成创建](/guide/res/create/4.png)
   
   ![命名领地](/guide/res/create/5.png)

## 领地管理命令

- `/res create <名称>` - 创建新的领地
- `/res remove <名称>` - 删除指定领地
- `/res list` - 查看您拥有的所有领地
- `/res help` - 查看更多帮助

## 注意事项

- 每个玩家可以创建多个领地，但总块数有限制
- 领地内的建筑和资源受到保护，默认其他玩家无法破坏，你可以自由编辑权限
- 创建领地需要收取雨点
