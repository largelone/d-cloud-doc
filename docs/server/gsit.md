# 动作插件

服务器安装了动作插件，提供多种有趣的交互动作。

## 坐下功能

### 坐在玩家头上

直接右键点击其他玩家，可以坐到对方的头上。

![坐在玩家头上](/guide/gsit/sit_on.png)

### 坐在半砖上

对着半砖右键点击也可以坐下。

![坐在半砖上 - 步骤1](/guide/gsit/sitclick-1.png)

![坐在半砖上 - 步骤2](/guide/gsit/sitclick-2.png)

## 特殊动作

### 旱地游泳姿势

输入命令 `/crawl` 可以做出旱地游泳的姿势。

![旱地游泳姿势](/guide/gsit/crawl.png)

### 躺下

输入命令 `/lanback` 或 `/lay` 可以让角色躺下。

![躺下姿势](/guide/gsit/layback.png)

### 旋转动作

输入命令 `/spin` cos钻头开始旋转

![旋转动作](/guide/gsit/spin.png)

## 使用说明

- 所有动作命令都可以在游戏中直接使用
- 右键交互功能无需输入命令
- 这些动作用于娱乐和社交互动，增加游戏趣味性
