# 核心规则

服务器规则旨在维护公平、友好的游戏环境。所有玩家必须遵守以下规则。

## 行为准则

### 1. 禁止作弊

以下行为严格禁止:

- 使用作弊客户端(X-ray、飞行、加速等)
- 利用游戏漏洞刷物品或金币
- 使用外挂脚本自动化操作
- 分享作弊工具或方法

违规处罚: 临时封禁 -> 永久封禁

### 2. 尊重他人

- 禁止在聊天中辱骂、歧视其他玩家
- 禁止发布垃圾信息或广告
- 禁止恶意骚扰或跟踪他人
- 禁止使用不适当的昵称

违规处罚: 警告 -> 禁言 -> 临时封禁

### 3. 保护他人财产

- 禁止破坏或修改他人建筑
- 禁止未经许可进入他人领地
- 禁止偷窃他人箱子物品
- 禁止恶意圈地影响他人

违规处罚: 赔偿 -> 临时封禁 -> 永久封禁

## 建筑规范

### 公共区域

- 特定区域建筑需符合主题风格
- 禁止建造不适当内容(色情、暴力、政治等)

### 私人领地

- 建筑高度无限制
- 禁止使用红石机器造成服务器卡顿
- 大型红石装置建议关闭或拆除


## 交易规则

- 禁止利用BUG进行不公平交易
- 禁止以现实货币交易游戏物品
- 禁止诈骗或欺骗其他玩家

违规处罚: 物品回收 -> 临时封禁

## 举报机制

发现违规行为可通过以下方式举报:

- QQ: 在举报频道提交证据
- 邮件: dcld@zhrhello.top

## 申诉流程

被处罚的玩家可以通过 QQ 联系管理团队申诉:

1. 私信提交申诉
2. 说明情况和理由
3. 管理员会重新审核
4. 申诉成功可解除处罚
5. 也可通过邮件申诉

::: info 提示
以上规则适用于所有服务器内成员，除非另有情况（如活动或特殊玩法等），否则应始终遵守它们。
:::
