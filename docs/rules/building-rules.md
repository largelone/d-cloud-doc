# 建筑规范

详细的建筑规则和指南。

## 建筑限制

### 允许内容

- 所有原版材料均可使用
- 允许红石电路和自动化装置
- 允许地下建筑
- 允许水上建筑

### 限制内容

- 高频红石装置
- 大规模刷怪塔(影响服务器性能)
- 超大型农场(超过100x100范围)
- 实体聚集装置

### 禁止内容

- 服务器卡顿装置(故意使TPS降低)
- 区块加载器(超过10个区块)
- 恶意陷阱

## 公共建筑指南

### 出生点周边(0-500格)

- 保持简洁美观
- 符合服务器主题风格
- 不建议建筑，可能会有管理员联系你拆除


## 私人领地建筑

### 建议

- 如果你做了很好的建筑，可以在群内分享，有机会纳入宣传视频，并被管理员免费加入全服地标哦！

### 领地边界

- 可以使用围栏标记边界
- 不要紧贴他人领地建造
- 保持至少5格缓冲距离

## 特殊建筑

### 农场

- 控制农场规模
- 使用高效而非大体积设计
- 避免使用实体聚集

!!! info 提示
    如果你真的想建造一个很有创意的建筑，但苦于我们的规定，请毫不犹豫地联系管理团队们！我们会根据实际情况对您的建筑重新进行审核并告知您是否可以进行建造。
