## 自定义警告高亮

除了标准的提示框，还可以使用自定义的警告高亮样式。在 Markdown 中使用 HTML 标签：

```html
<div class="highlight-warning">

**重要提醒**：这是使用自定义警告高亮的示例。这种样式更加醒目，适合特别重要的警告信息。

- 第一点注意事项
- 第二点注意事项
- 第三点注意事项

</div>
```

实际效果如下：

## 代码块

### 行内代码

使用反引号包裹行内代码，例如：`/reg <密码> <确认密码>`

### 代码块

```bash
# 注册账号
/reg 123456 123456

# 登录账号
/l 123456
```
