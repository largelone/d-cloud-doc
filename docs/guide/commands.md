# 基础命令

本页面列出服务器常用的基础命令。

## 账号相关

| 命令 | 说明 | 示例 |
|------|------|------|
| `/regr <密码> <密码>` | 注册账号 | `/reg 114514 114514` |
| `/l <密码>` | 登录账号 | `/l 114514 114514` |
| `/changepassword <旧密码> <新密码>` | 修改密码 | `/changepassword 114514 1919810` |

## 经济相关

| 命令 | 说明 | 示例 |
|------|------|------|
| `/money` 或 `/balance` | 查看余额 | `/money` |
| `/money pay <玩家> <金额>` | 转账给其他玩家 | `/money pay Steve 100` |
| `/baltop` | 财富排行榜 | `/baltop` |
| `/market` | 打开本区市场 | `/market` |

## 传送相关

| 命令 | 说明 | 示例 |
|------|------|------|
| `/spawn` | 返回主广场 | `/spawn` |
| `/home <名称>` | 设置家 | `/home base` |
| `/home <名称>` | 传送到家 | `/home base` |
| `/homes` | 查看家列表 | `/homes` |
| `/homes rm <名称>` | 删除家 | `/homes rm base` |
| `/tpa <玩家>` | 请求传送 | `/tpa Steve` |
| `/tpyes` | 接受传送 | `/tpyes` |
| `/tpno` | 拒绝传送 | `/tpno` |
| `/back` | 返回上次位置 | `/back` |

## 社交相关

| 命令 | 说明 | 示例 |
|------|------|------|
| `/msg <玩家> <内容>` | 私聊 | `/msg Steve Hello` |
| `/afk` | 设置离开状态 | `/afk` |

::: warning 提示
    在实际游戏中，可以通过在聊天内容前加上"!"来达到“全服喊话”的目的。
    假设我要发送"123"，则应输入："!123"。

## 服务器信息

| 命令 | 说明 | 示例 |
|------|------|------|
| `/list` | 查看在线玩家 | `/list` |

## 提示

- 大多数命令可以通过 `/help <命令>` 查看详细用法
- 命令可以在聊天框中按 `Tab` 键自动补全
- 如果命令不生效,请检查是否有足够权限
