# 常见问题

服务器相关的常见问题解答。

## 连接问题

### Q: 无法连接服务器?

**A:** 按以下顺序排查:

1. 确认Minecraft版本在 1.20 - 1.20.4 之间
2. 检查网络连接是否稳定
3. 尝试备用地址 <span class="copy-inline" data-copy="mc.d-cloud.com" title="点击复制">`mc.d-cloud.com`</span>
4. 查看QQ公告确认服务器状态
5. 重启Minecraft和路由器

### Q: 连接时显示 "Outdated Client"?

**A:** 你的Minecraft版本过低。请升级到 1.20.x 版本。

### Q: 连接时显示 "Outdated Server"?

**A:** 服务器正在更新中,请稍候再试。

### Q: 延迟很高怎么办?

**A:** 

- 选择离你地理位置较近的服务器区域
- 关闭占用带宽的应用
- 使用有线网络连接
- 避免使用WiFi

## 账号问题

### Q: 忘记密码怎么办?

**A:** 

1. 使用注册邮箱找回
2. 在QQ联系管理员
3. 提供登录IP和注册时间验证身份

### Q: 可以修改密码吗?

**A:** 可以,使用命令:
```
/changepassword <旧密码> <新密码>
```

### Q: 可以修改昵称吗?

**A:** 可以联系管理员修改,需要有合理理由。

## 游戏问题

### Q: 如何获得起始资金?

**A:** 注册后自动获得 500 金币。

### Q: 如何保护自己的领地?

**A:** 使用领地系统:
```
/res create <领地名称>
```
然后使用木铲点击两个对角划定区域。

### Q: 如何传送回家?

**A:** 
```
/home set <名称>    # 先设置家的位置
/home <名称>        # 传送回家
```

### Q: 可以飞行吗?

**A:** 

- 创造模式飞行: 不允许
- 鞘翅飞行: 允许,需要安装鞘翅模组
- 创造权限: 仅限管理员

### Q: 服务器有地图边界吗?

**A:** 

- 世界边界: 30,000 格
- PVP安全区: 出生点 500 格内
- 其他区域可自由探索

## 交易问题

### Q: 如何与其他玩家交易?

**A:** 

- 面对面交易: `/trade <玩家名>`
- 直接转账: `/pay <玩家名> <金额>`
- 通过商店交易

### Q: 被骗了怎么办?

**A:** 

1. 保留交易记录截图
2. 立即联系管理员
3. 提供详细情况说明
4. 管理员会根据情况处理

## 举报与申诉

### Q: 如何举报违规玩家?

**A:** 

- 游戏内: `/report <玩家名> <原因>`
- QQ: 在举报频道提交证据
- 邮件: support@d-cloud.mc

### Q: 被误封了怎么办?

**A:** 

1. 在QQ提交申诉
2. 说明情况和理由
3. 提供相关证据
4. 等待管理员审核

### Q: 申诉需要多长时间?

**A:** 通常在 24-48 小时内处理。

## 其他问题

### Q: 服务器会重置吗?

**A:** 服务器地图永不重置。你的所有进度都会保留。

### Q: 可以安装模组吗?

**A:** 

- 允许: 小地图、状态显示、辅助UI
- 禁止: 任何提供不公平优势的模组

### Q: 服务器在线人数上限是多少?

**A:** 最大支持 200 人同时在线。

### Q: 如何联系管理员?

**A:** 

- QQ: 最直接的方式
- 游戏内: 留言或举报系统
- 邮件: support@d-cloud.mc
